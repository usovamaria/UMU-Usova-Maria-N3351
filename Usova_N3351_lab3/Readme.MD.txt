Инструкция по запуску программного модуля к Лабораторной работе №3

1. Поместите в одну директорию (папку):
	1) файл module.py - файл программного модуля (сейчас находится в папке "входные и выходные данные")
	2) файл FreeSans.ttf - файл шрифта, используемого для формирования текста в pdf на русском языке (сейчас находится в папке "входные и выходные данные")
2. Для запуска программмного модуля в консоли, находясь в директории, введите "python3 module.py"
3. Для просмотра выходных данных:
	1) Программой автоматически сформируется выходной файл "bill.pdf" (сейчас он также находится в папке "входные и выходные данные")
	2) Для просмотра откройте данный файл

Примечание: входные данные для формирования счета на оплаты являются частью программного модуля. 
При необходимости их можно изменить.
