Инструкция по запуску программного модуля к Лабораторной работе №2

0. Подготовительный этап.
	1) В режиме консоли перейдите в директорию (папку), в которой находится файл "nfcapd.202002251200"
	2) В консоли введите следующую команду: nfdump -r nfcapd.202002251200 -o csv
	Данная команда преобразует файл в формат csv.
	3) В любом редакторе таблиц (MO Excel, Google таблицы) произведите сортировку данных по первому столбцу по возрастанию (дата и время возрастают с номером строки)
1. Поместите в одну директорию (папку):
	1) файл с исходными данными формата *.csv, назовите его data.csv (файл, который сейчас находится в папке "входные и выходные данные" преобразован в нужный формат и готов к анализу)
	2) файл bill.txt для выходных данных
	3) файл module.py - файл программного модуля (сейчас находится в папке "входные и выходные данные")
2. Для запуска программмного модуля в консоли, находясь в директории, введите "python3 module.py"
3. Для просмотра выходных данных:
	1) Программой автоматически откроется график зависимости объема трафика от времени; 
	2) Закройте файл с графиком после просмотра;
	3) Откройте файл "bill.txt" с помощью любого текстового редактора для просмотра результатов тарификации.
